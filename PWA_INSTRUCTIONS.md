# PWA версия приложения Misbaha

## Что такое PWA?

Progressive Web App (PWA) — это веб-приложение, которое можно установить на главный экран устройства и использовать как нативное приложение. Работает на iPhone без Apple Developer аккаунта!

## Преимущества PWA:

✅ **Бесплатно** — не требует Apple Developer аккаунта  
✅ **Работает на iPhone** — можно добавить на главный экран  
✅ **Работает офлайн** — сохраняет данные локально  
✅ **Быстрое** — работает как нативное приложение  
✅ **Не требует установки** — просто откройте в браузере  

## Как использовать PWA версию:

### 1. Запуск для разработки:

```bash
npm run web
```

Откройте браузер по адресу `http://localhost:8081` (или адресу, который покажет Expo)

### 2. Сборка для production:

```bash
npm run build:web
```

Это создаст папку `dist` с готовым PWA приложением.

### 3. Размещение на сервере:

Загрузите содержимое папки `dist` на любой веб-хостинг:
- GitHub Pages (бесплатно)
- Netlify (бесплатно)
- Vercel (бесплатно)
- Ваш собственный сервер

### 4. Установка на iPhone:

1. Откройте сайт в Safari на iPhone
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
3. Выберите "На экран «Домой»"
4. Приложение установится на главный экран

### 5. Установка на Android:

1. Откройте сайт в Chrome на Android
2. Появится баннер "Установить приложение"
3. Нажмите "Установить"
4. Или откройте меню (три точки) → "Установить приложение"

## Локальное тестирование:

После сборки можно протестировать локально:

```bash
npm run serve:web
```

Откройте `http://localhost:3000` в браузере.

## Особенности PWA версии:

- ✅ Все функции работают как в мобильном приложении
- ✅ Данные сохраняются локально (localStorage вместо AsyncStorage)
- ✅ Работает офлайн после первой загрузки
- ✅ Вибрация заменена на звуковые сигналы для веба
- ✅ Можно добавить на главный экран

## Ограничения PWA:

- ⚠️ Вибрация работает только на устройствах, которые поддерживают Web Vibration API
- ⚠️ Некоторые функции могут отличаться от нативной версии
- ⚠️ Требует интернет для первой загрузки

## Размещение на GitHub Pages (бесплатно):

1. Создайте репозиторий на GitHub
2. Соберите проект: `npm run build:web`
3. Скопируйте содержимое `dist` в репозиторий
4. Включите GitHub Pages в настройках репозитория
5. Ваше приложение будет доступно по адресу: `https://ваш-username.github.io/репозиторий`

## Размещение на Netlify (бесплатно):

1. Создайте аккаунт на https://netlify.com
2. Соберите проект: `npm run build:web`
3. Перетащите папку `dist` на сайт Netlify
4. Получите ссылку на ваше приложение

## Размещение на Vercel (бесплатно):

1. Установите Vercel CLI: `npm i -g vercel`
2. Соберите проект: `npm run build:web`
3. Перейдите в папку `dist`
4. Запустите: `vercel`
5. Следуйте инструкциям

## Файлы PWA:

- `web/manifest.json` - манифест приложения (название, иконки, цвета)
- `app.json` - конфигурация Expo с настройками для веба
- `dist/` - собранное приложение (создается после `npm run build:web`)

## Обновление приложения:

После изменений пересоберите проект:
```bash
npm run build:web
```

И загрузите новую версию на сервер.

# Варианты сборки iOS приложения без Apple Developer

## ⚠️ Важно знать

**Apple требует Apple Developer аккаунт ($99/год) для установки приложений на реальные iOS устройства.**

Однако есть несколько вариантов:

---

## Вариант 1: iOS Симулятор (БЕСПЛАТНО, но только на Mac)

Можно запускать приложение в iOS симуляторе на Mac без Apple Developer аккаунта.

### Требования:
- Mac компьютер
- Xcode (бесплатно из App Store)
- CocoaPods

### Шаги:

1. **Установите Xcode:**
   - Откройте App Store на Mac
   - Найдите и установите Xcode

2. **Установите CocoaPods:**
```bash
sudo gem install cocoapods
```

3. **Соберите проект:**
```bash
cd ios
pod install
cd ..
npx expo run:ios
```

Это запустит приложение в iOS симуляторе на вашем Mac.

---

## Вариант 2: Бесплатный Apple ID (Ограниченная установка)

Apple позволяет использовать бесплатный Apple ID для установки приложений на свое устройство, но с ограничениями:

### Ограничения:
- ✅ Можно установить только на свои устройства
- ✅ Приложение работает только 7 дней
- ✅ После 7 дней нужно переустанавливать
- ✅ Максимум 3 приложения одновременно
- ❌ Нельзя публиковать в App Store
- ❌ Нельзя использовать TestFlight

### Шаги:

1. **Подготовка проекта:**
```bash
npx expo prebuild
```

2. **Откройте проект в Xcode:**
```bash
cd ios
open misbaha.xcworkspace
```

3. **В Xcode:**
   - Выберите ваш проект в навигаторе
   - Выберите Target "misbaha"
   - Перейдите в "Signing & Capabilities"
   - Включите "Automatically manage signing"
   - Выберите ваш Team (ваш Apple ID)
   - Xcode автоматически создаст временный профиль

4. **Подключите iPhone:**
   - Подключите iPhone к Mac через USB
   - Выберите ваше устройство в Xcode
   - Нажмите Run (▶️)

5. **На iPhone:**
   - Перейдите в Настройки → Основные → Управление устройством
   - Доверьтесь разработчику (вашему Apple ID)

**⚠️ Приложение будет работать только 7 дней, затем нужно переустанавливать.**

---

## Вариант 3: Apple Developer Program ($99/год) - РЕКОМЕНДУЕТСЯ

Это единственный способ для полноценной установки и распространения:

### Преимущества:
- ✅ Установка на любое количество устройств
- ✅ Приложение работает бессрочно
- ✅ Возможность публикации в App Store
- ✅ Доступ к TestFlight для бета-тестирования
- ✅ Push уведомления и другие функции

### Шаги:

1. **Купите Apple Developer аккаунт:**
   - https://developer.apple.com/programs/
   - $99 в год

2. **Соберите через EAS Build:**
```bash
eas build --platform ios --profile production
```

3. **Или локально через Xcode:**
   - Откройте проект в Xcode
   - Выберите ваш Apple Developer Team
   - Соберите и установите

---

## Вариант 4: Использование Expo Go (Текущий вариант)

Если вы просто хотите использовать приложение:

```bash
npm start
```

Затем откройте Expo Go на iPhone и отсканируйте QR-код.

**Плюсы:** Работает сразу, бесплатно  
**Минусы:** Требует Expo Go, не standalone приложение

---

## Рекомендация

Для личного использования на своем iPhone:
1. **Если у вас Mac** → Используйте Вариант 2 (бесплатный Apple ID, но нужно переустанавливать каждые 7 дней)
2. **Если нет Mac** → Используйте Expo Go или купите Apple Developer аккаунт и используйте EAS Build

Для публикации в App Store:
- Обязательно нужен Apple Developer аккаунт ($99/год)

---

## Альтернатива: PWA (Progressive Web App)

Если хотите обойти App Store полностью, можно создать PWA версию приложения:

1. Добавьте поддержку веб-версии
2. Пользователи могут добавить сайт на главный экран iPhone
3. Работает как нативное приложение

Но это не полноценное iOS приложение и требует доработки кода для веба.
